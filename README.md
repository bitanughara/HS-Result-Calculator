# 🎓 HS Result Calculator

A menu-driven Java console application that calculates WBCHSE Higher Secondary (HS) results using the **Best of 5** rule.

## ✨ Features

- Accepts marks for 6 subjects
- Validates marks (0–100)
- Calculates Best of 5 total marks
- Calculates percentage
- Displays Grade and Result
- Menu-driven interface
- Simple and beginner-friendly Java project

## 🛠️ Technologies Used

- Java
- Scanner Class
- if-else Statements
- do-while Loop

## 📌 How It Works

1. Enter marks for:
   - Bengali
   - English
   - Physics
   - Chemistry
   - Mathematics
   - Optional Subject
2. The program removes the lowest mark among Physics, Chemistry, Mathematics, and Optional Subject.
3. Calculates:
   - Best of 5 Total Marks (Out of 500)
   - Percentage
   - Grade
4. Displays the final result.
